# gpt2.0 (nanoGPT 기반 모델)

### 1. 프로젝트 개요
* 본 프로젝트는 nanoGPT 아키텍처를 활용하여 신데렐라 동화를 학습하고, 문맥을 이해하여 새로운 텍스트를 생성하는 모델을 구현함.

### 2. 프로젝트 구조
* `data/my_dataset/input.txt`: 학습 데이터(신데렐라 동화).
* `out-my-dataset/ckpt.pt`: 학습 완료된 모델 파라미터.
* `train.py`: 모델 학습 스크립트.
* `sample.py`: 텍스트 생성 스크립트.

### 3. 성능 평가
* **학습 설정**: 500 iterations, batch_size 64, block_size 256.
* **Loss 변화**: 
    - step 0: train loss 5.9878
    - step 500: train loss 0.0200 (성공적인 학습 완료)

### 4. 생성 샘플 결과
* 입력: "옛날"
* 결과: "옛날, 왕궁으로 떠났습니다. 홀로 남은 신데렐라는 눈물을 흘렸습니다. “나도 한 번쯤 아름다운 옷을 입고 춤을 춰 보고 싶었는데….” 바로 그 순간, 눈부신 빛이 부엌 안을 가득 채웠습니다..."
